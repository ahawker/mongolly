# Mongolly

**Easy backups for EBS-based MongoDB Databases**

More details coming soon.

## Installation

Add this line to your application's Gemfile:

    gem 'mongolly'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongolly

## Usage

`mongolly help usage`
